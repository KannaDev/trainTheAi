Who we are

We are a couple of High School graduates that want to see what the whole Ai fuss is about. We are making A.I. powered web applications for the user to interact and make it better. We have never done anything like this before, so everything might not be perfect.

We build simple web games like Tic-tac-toe and Rock-paper-scissors that have a A.I neural network system attached in the backend so the program could imitate a human player.

We mainly use Python3 with libraries like numPy. We also use Java in certain cases, really depends on the type of application we are building. We know we are no professionals/experts in this area, that's why the whole code is open-source on github

We are software developers. Building programs is what we are good at. Since we are only High school graduates at the moment, we have to take baby steps and build our own projects one by one for our resume/portfolio. We love what we do, and this is our passion.
